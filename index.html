Implementation of algorithms to compute the geodesic distance between phylogenetic trees

Abstract


Comparing and computing distances between phylogenetic trees are important biological problems, especially for models where edge lengths play an important role. A space modeling the set of all phylogenetic trees having a fixed set of leaves is considered. This space provides a way of measuring distance between phylogenetic trees. It also provides a justification for disregarding portions of a collection of trees that agree, thereby, simplifying the space in which comparisons are to be made.

The geodesic distance measure between two phylogenetic trees with edge lengths is the length of the shortest path between them in this continuous tree space. We aim to understand and implement different algorithms that provide a way to compute geodesic distances in tree space. Specifically, we would be looking at three different algorithms including the GTP algorithm, GeodeMaps and GeoMe Tree.

Plan of Action


  • Study and understand the geometry of the space of phylogenetic trees.
  • Understand and implement the algorithms - GTP, GeodeMap and GeoMeTree that help compute the geodesic distance between phylogenetic trees.
  • Draw comparisons between these algorithms.


Dataset Used


The dataset being used for the implementation is TreeBASE. It is a repository containing all phylogenetic information, phylogenetic trees and the data needed to generate them.


Work Distribution


Since we are going to deal with three algorithms, the workload will be equally split among the team members.
